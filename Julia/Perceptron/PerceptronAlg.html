In machine learning, the perceptron is an algorithm for supervised learning of binary classifiers. A binary classifier is a function which can decide whether or not an input, represented by a vector of numbers, belongs to some specific class.This algorithm is often refered to as the ''Hello World'' of machine learning

Note**:There are two types of Perceptrons:Single Layer & Multilayer. Single layer perceptrons are only capable of learning linearly separable patterns.
