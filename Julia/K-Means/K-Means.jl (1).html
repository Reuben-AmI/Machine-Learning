K-Means clustering

K-means is an Unsupervised learning algorithm where vectors are grouped according to their features.K-means clustering is an unsupervised learning algorithm that aims to partition n observations into k clusters in which each observation belongs to the cluster with the nearest mean (cluster centers or cluster centroid).Centroids try to find the center of clusters

In this notebook we will implement K-means on artifical data

The steps of K-mean alg are:

  1. Caculate length of each feature vector to each centroid.For this notebook we will use the euclydian distance formula to caculate distance

  2. Assign each feature vector the centroids that it is closet too.

  3. Update the centroids by calculating the mean of along the jth-dimension for each cluster

  4. Repeat step until the stopping condition is met.
